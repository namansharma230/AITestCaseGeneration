💡 Troubleshooting Tips
  • Make sure Edge is running with remote debugging — double-click start_edge.bat
  • Ensure you are logged into Jira / Confluence in that Edge window before submitting
  • Confluence only: after logging in, manually open the Confluence page once in Edge, then re-submit the URL here
  • Verify your Groq API key is correct in the .env file
  • Check that the URL is accessible (not behind a VPN or firewall)
  • Try refreshing the Jira / Confluence page manually in Edge and resubmitting
